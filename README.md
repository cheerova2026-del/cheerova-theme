# Cheerova theme

Custom Shopify theme for **Cheerova**, built on [Dawn](https://github.com/Shopify/dawn), Shopify's reference theme.

> ⚠️ This repo is connected to the store via Shopify's GitHub integration.
> Every commit to `main` automatically updates the theme **cheerova-theme/main** in the Theme library.

## Workflow

1. Claude writes and edits the code, step by step
2. Changes are committed to this repo
3. Shopify syncs them to the unpublished theme within seconds
4. We preview, then approve or adjust
5. When everything is ready → publish

## Rules we follow

- The live store is never touched until launch day
